How to pay for mobile operator bill

1. Open " ACLEDA MOBILE " app

2. select " PAYMENT " to menu

3. Use Touch ID, Facial, or enter PIN to login

4. Select "Mobile Operator"

5. Select mobile operator company

6.1. Enter payment information

  • Select debit account
  • Input customer id
  • Input payment amount

6.2. Choose "OK" button

7.1. Verify payment information

7.2. Choose "Confirm" button

8. Payment Completed
