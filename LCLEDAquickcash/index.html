Quick Cash Withdrawal by Code (Cardless)

  1. 1. Open ACLEDA mobile app
  2. 2. Select "Quick Cash" menu
3. Input PIN, Facial, or Touch ID
4. Select "CLOSE" button
  1. 5. Select debit account
  2. 6. Input withdrawal amount
  3. 7. Choose "OK" button
8. You will get a 6-digit number to enter at ATM to withdraw money (this code is valid for 24 hours only)
  1. 9. Please click on any button or ATM screen to start
  2. 10. Select language
11. Select "Quick Cash by Code" menu
  1. 12. Enter the code generated on ACLEDA mobile
  2. 13. Press "Yes" button
  1. 14. Enter the withdrawal amount the same as the amount created on ACLEDA mobile
  2. 15. Press "Yes" button
16. Please take your money from ATM
